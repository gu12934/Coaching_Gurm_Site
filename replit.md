# Personality & Career Discovery Website

## Project Overview
A static HTML/CSS/JS website for personality assessment resources, astrological insights, and career coaching services. Designed for deployment on GitHub Pages.

## Purpose
Provide a comprehensive platform for users to:
- Discover free personality and career assessments
- Learn about INFJ personality type and astrological influences
- Access career coaching and guidance services
- Get birth chart interpretations and astrology resources
